# Interactive Romantic Proposal Web Page 💖

A beautifully designed, customizable, and interactive romantic proposal webpage. This project features a nostalgic grid-paper aesthetic where users can drag and drop paper elements to reveal hidden messages and a final surprise. 

## ✨ Features

* **Drag and Drop Interface:** Smoothly move paper notes across the screen for a fun, interactive experience.
* **Paper Aesthetic:** Designed to look like a digital scrapbook or a handmade love letter.
* **Fully Customizable Text:** Every romantic line and quote on the web page can be easily modified to suit your personal message.
* **Easy Image Swapping:** The template comes with placeholder images, which can be instantly replaced with your own memories and photos.

## 🚀 How to Customize

Follow these simple steps to personalize the webpage for your special someone:

1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)

2.Change the Images:

   <> Navigate to the folder where images are stored.

   <> Replace the existing placeholder photos (currently Elsa) with your own images.

   <> Make sure your new images match the exact filenames in the code, or update the image paths directly in the index.html file.

3.Customize the Text:

   <> Open the index.html file in any code editor (like VS Code).

   <> Locate the text elements (e.g., "You are Cute", "Amazing ❤️") and replace them with your own personalized messages.

4.Test it Locally:

   <> Simply open the index.html file in any web browser to see your customized proposal in action!
